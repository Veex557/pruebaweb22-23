A travers d'aquesta pàgina de prova, que és l'index.html de ProvaWeb (la primera web que feim a l'assignatura d'Aplicacions Web), veurem distintes coses.

Estructura de una web

El codigo de una web (como conjunto de diferentes paginas) se guarda dentro de una carpeta con el nombre paginas.

Tendra los diferentes directorios:

estilo: Tiene codigos css para editar el estilo y pintar nuestra pagina web

imagens: Tiene imagenes de la web.

paginas: Tiene el codigo html de toda la pagina web.

Tambien esta la pagina indice (Es la actual)en la que encontramos acceso a todas las paginas.

Estructura de un documento HTML

Un fichero HTML tiene que tener la siguiente estructura

Una etiqueta Html y dentro dos etiquetas: header i body.
